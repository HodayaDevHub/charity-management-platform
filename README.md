# Charity Management Platform

A robust, full-stack web application designed to streamline charitable donations, fundraiser management, and community support tracking. This platform offers an intuitive ecosystem for administrators, fundraisers, and donors to collaborate efficiently.

---

## 🚀 Tech Stack

### Frontend
* **React.js** (Functional Components, Hooks)
* **Vite** (Build tool for optimized performance)
* **CSS3** (Responsive and modern UI design)

### Backend & Database
* **Node.js** & **Express.js** (RESTful API architecture)
* **MongoDB** & **Mongoose** (NoSQL database management and data modeling)
* **JWT (JSON Web Tokens)** (Secure, stateless user authentication)

---

## ✨ Key Features

* **Secure Authentication:** Role-based access control and secure login for Administrators, Fundraisers, and Donors using JWT.
* **Fundraiser Management:** Dynamic creation, tracking, and updating of ongoing charity campaigns.
* **Donation Tracking:** Live calculation of amount raised, keeping donors informed about campaign progress.
* **Group Coordination:** Ability to organize community donation drives and team-based fundraising goals.
* **Responsive Dashboard:** A clean user interface optimized for both desktop and mobile devices.

---

## 📂 Architecture Overview

The repository is divided into two main components:

* `/client` - The frontend React application built with Vite.
* `/server` - The backend Node.js/Express API handling data storage, business logic, and authentication middleware.
